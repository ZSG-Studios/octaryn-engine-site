Octaryn Engine is access-controlled for verified contributors.

It is not open source.

Verified contributors may inspect the repository, clone it, build locally,
modify locally, and submit pull requests to the official Octaryn repositories.

You may not redistribute the source, publish forks or mirrors, distribute
binaries, reuse repository content elsewhere, or commercialize the project
unless Zachary David Rose separately authorizes it in writing.

Before your first substantive pull request, complete the contributor intake
form:

https://script.google.com/macros/s/AKfycbyGsHf7AzW0qI_kUaegpCnmitAFibEJpZmuFQYInYjenw40W7BExqRMsYSe-QW4r1U4/exec

Start from the public intake page if you need the public reading summaries first:

https://zsg-studios.github.io/octaryn-engine-site/contribute.html
